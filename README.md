## Rules

* UI/UXを中心に見る
* アプリの内容自体の良し悪しには触れない
* ゲーム系アプリを除く
* エラー時（オフライン時）の表示に興味あるかも
* ゆるく
* 以下の項目を記載する
  * バージョン
  * 使用端末
  * 使っていそうなUIKit
  * 所感

⚠️：ルールは随時変更の可能性あり
