# 🎬 TED

* OS 9.2.1 / App 5.0.2
* iPhone6

### Purpose
* TEDのトーク動画を閲覧

### UI / UX  
* 起動時のチュートリアルなどは特になし
* **一覧 > 詳細 > 動画再生画面の王道フロー**かつシンプルな構成でわかりやすい
* タブ構成だが、[どのタブから遷移しても**終着地は動画再生画面**](#ted_main)（アプリの目的に沿った画面フロー）
* オフラインでもコンテンツの一覧表示はあるが、動画の再生が行われない
