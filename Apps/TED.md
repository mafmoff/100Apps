# 🎬 TED

* OS 9.2.1 / App 5.0.2
* iPhone6

### Purpose
TEDのトーク動画を閲覧

### UI / UX  
* 起動時のチュートリアルなどは特になし
* **一覧 > 詳細 > 動画再生**の王道フロー、シンプルな構成でわかりやすい
* タブ構成だが、[どのタブから遷移しても終着地は動画再生画面](#ted_nav)（アプリの目的に沿った画面フロー）
* オフラインでもコンテンツの一覧表示はあるが、動画の再生が行われない
* TED配信の動画は表示が[リッチなパララックス](#ted_richParallax)でボタンとして画面上部に固定になるような動きがある
